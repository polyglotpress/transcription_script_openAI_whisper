# Transcribe a YouTube video #

This is a my first Python script where the user can download an audio file (.wav) and its transcript in a .txt file by simply pasting a YouTube video link when prompted in the terminal.

## Requirements ##
- ffmpeg
- python

## How to run ##

1. Clone this project
2. Ensure you have ffmpeg installed on your system and added to your PATH
3. Open a terminal in the project folder
4. Paste ` pip install -r requirements.txt `
5. When that is complete, run `python transcription.py`
6. Follow the prompts and you'll see a new folder with the downloads on your desktop.
